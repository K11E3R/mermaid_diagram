How It Works

Creating diagrams is simple and intuitive. Follow these three easy steps:

1

Insert a Code Block

Type /code in any Knowledge article or HTML field to insert a code block. The command menu will appear instantly.

2

Select Mermaid Language

Click the language dropdown (shows "Plain Text" by default) and select Mermaid from the comprehensive list of programming languages.

3

Write and Render

Enter your Mermaid syntax and watch your diagram render automatically! Click on the diagram anytime to edit it. Changes are saved automatically with your article.

Installation

  1. Download or clone the module into your Odoo addons directory
  2. Restart your Odoo server
  3. Go to Apps → Update Apps List
  4. Search for "Mermaid Diagrams"
  5. Click Install
  6. Start creating diagrams immediately - no configuration needed!

Technical Information

Dependencies

  • html_editor (included in Odoo 19 core)
  • Mermaid.js 11.x (loaded from CDN)

Compatibility

  • Odoo 19.0 Community and Enterprise
  • Works on Odoo.sh, Odoo Online, and On-Premise
