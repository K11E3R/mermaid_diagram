How to Use

Step 1: Insert Code Block

Type /code in any Knowledge article or HTML editor to insert a code block. The command menu will appear instantly with various options.

Step 2: Select Mermaid

Click the language dropdown and choose Mermaid from the list. Native integration with Odoo's syntax highlighting system.

Step 3: Create Your Diagram

Write your diagram syntax and watch it render in real-time. Click any diagram to edit. Changes are saved automatically.

Installation

  1. Deploy module to Odoo addons directory
  2. Restart Odoo server instance
  3. Navigate to Apps and update apps list
  4. Search for Mermaid Diagrams
  5. Click Install button
  6. Begin creating diagrams immediately
