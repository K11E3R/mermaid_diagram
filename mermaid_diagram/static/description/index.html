How to Use

Step 1: Insert Code Block

Type /code in any Knowledge article to insert a code block.

Step 2: Select Mermaid

Choose Mermaid from the language dropdown.

Step 3: Write Your Diagram

Enter your diagram code and watch it render in real-time.

Supported Diagram Types

Powered by Mermaid.js 11.x

Flowcharts | Sequence Diagrams | Gantt Charts | ER Diagrams | State Diagrams | Class Diagrams | Pie Charts | User Journey | Git Graphs | Mind Maps

flowchart TD

Compatibility

Chrome 90+, Firefox 88+, Safari 14+, Edge 90+.

Installation

  1. Deploy module to Odoo addons directory
  2. Restart Odoo server
  3. Go to Apps and update apps list
  4. Search for Mermaid Diagrams
  5. Click Install
